# InvestED Project Guide

This repo is a Vite + React + Tailwind single-page site for the InvestED fund. Below is a quick map of folders and key files, and what each is responsible for.

## Top-Level
- `package.json` / `package-lock.json`: Project metadata, scripts, dependencies.
- `vite.config.js`: Vite build/dev config.
- `postcss.config.js`: PostCSS pipeline (used by Tailwind).
- `tailwind.config.js`: Tailwind theme and content paths.
- `index.html`: Vite entry HTML for development.
- `README.md`: Project overview (if populated).
- `dist/`: Production build output (generated by Vite).

## src (application code)
- `main.jsx`: App bootstrap; mounts React into the DOM.
- `App.jsx`: App shell; wraps routes with navigation and footer.
- `index.css`: Tailwind base imports and global styles (body background, typography).

### src/components
- `Navigation.jsx`: Top nav bar with active route styling.
- `Footer.jsx`: Site footer with links and brand text.
- `Icons.jsx`: Centralized SVG icon components used across pages.

### src/pages
- `Homepage.jsx`: Landing page with hero, stats, fund basics, progress visuals, navigation cards, and educational pitch.
- `About.jsx`: Fund overview; mission, purpose, student roles, governance flow, decision-making steps, and finance concepts. Includes a colorful “spotlight” header grid.
- `Charter.jsx`: Fund Charter content; download CTA, purpose, objectives, responsibilities, ethics, and FAQ. Includes a “charter snapshot” highlight grid.
- `IPS.jsx`: Investment Policy Statement; overview, download CTA, goals, decision principles, philosophy, and risk education. Includes an “IPS at a glance” highlight grid.
- `TrainingPathway.jsx`: Training program; overview, session breakdowns, resources, learning outcomes, and finance concepts. Includes a “training highlights” grid.

### dist (generated)
- `index.html`, `assets/index-*.css/js`: Bundled production assets produced by Vite (no need to edit; regenerate via build).
